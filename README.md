# dc-date-and-time  
現在の日付と時刻を表示してクリップボードにコピーするだけのスクリプト。
## 使い方  
1. jQueryを読み込む。
2. dc-date-and-time.jsを読み込む。
3. 以下のHTMLを記述する。
```
<div class="dcdt-wrap">
    <form>
        <select class="dcdt-select"></select>
        <button class="dcdt-cpy" type="button">コピー</button>
        <button class="dcdt-ref" type="button">更新</button>
    </form>
</div>
```
